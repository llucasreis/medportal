## Features Implementadas

- Backend:
  - Autenticação de usuário por JWT
  - Criação de usuário
  - Criação de médico
  - Criação de agendamento
  - Listagem de médicos
  - Listagem de agendamentos

- Frontend:
  - Login
  - Cadastro de usuário
  - Listagem de médico
  - Cadastro de médico
  - Listagem de agendamentos
